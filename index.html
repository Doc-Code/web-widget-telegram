Демонстрация Telegram виджета

Этот виджет позволяет добавить на ваш сайт удобную кнопку для связи через Telegram. Пользователю достаточно кликнуть на иконку или текстовый пузырь, чтобы перейти в чат с вашим ботом.

Три способа установки и настройки

1. Простая установка

Просто скопируйте весь код и вставьте его в HTML-страницу прямо перед закрывающим тегом </body>: 

<script>
// Вставьте здесь весь код виджета
</script>

2. Настройка через data-атрибуты

Скопируйте файл simple-telegram-widget.js на свой сервер и подключите его с настройками через data-атрибуты: 

<script 
  src="simple-telegram-widget.js"
  data-telegram-username="your_bot_username"
  data-title="Бот бронирования"
  data-subtitle="и общения на тему проживания..."
  data-color="#0088cc"
  data-position="bottom-right"
  data-wave-strength="strong"
  data-dark-mode="false"
  data-gradient-border="true"
></script>

3. Настройка через JavaScript

Настройте виджет после загрузки страницы с помощью JavaScript:

<script src="simple-telegram-widget.js"></script>
<script>
  configureTelegramWidget({
    telegramUsername: 'your_bot_username',
    title: 'Бот бронирования',
    subtitle: 'и общения на тему проживания...',
    color: '#0088cc',
    borderColor: '#0088cc',
    position: 'bottom-right',
    waveStrength: 'strong',
    attentionEffect: 'bounce',
    darkMode: false,
    gradientBorder: true
  });
</script>

Демонстрация возможностей

Демонстрация тёмной темы

Виджет автоматически адаптируется к тёмному фону вашего сайта.

Все доступные настройки

Параметр Описание Значение по умолчанию
telegramUsername Имя пользователя Telegram бота (без @) your_bot_username
title Заголовок в пузыре Бот бронирования
subtitle Подзаголовок в пузыре и общения на тему проживания...
position Положение виджета на странице bottom-right
showDelay Задержка появления пузыря после загрузки (мс) 1000
hideDelay Время до скрытия пузыря (мс) 10000
waveDelay Задержка до начала анимации (мс) 15000
waveInterval Интервал между анимациями (мс) 30000
waveStrength Сила махания (light, medium, strong) medium
attentionEffect Тип анимации (wave, bounce, glow, none) wave
color Основной цвет виджета #0088cc
borderColor Цвет рамки пузыря #0088cc
titleColor Цвет заголовка #0088cc
subtitleColor Цвет подзаголовка #555
autoOpen Автоматически показывать пузырь true
rememberState Запоминать закрытие пользователем false
darkMode Использовать тёмную тему false
gradientBorder Использовать градиентную рамку false
zIndex Z-index контейнера виджета 9999
